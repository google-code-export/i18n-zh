2. Validate, correct, and pretty HTML file

2.1. tidy

$ sed /^[[:space:]]*$/d 40.html > tmp
$ tidy -utf8 -b -i -q --wrap 0 --output-bom no --newline LF -asxhtml \
      --tidy-mark no --doctype transitional -o example.html tmp
$ /bin/rm -f tmp

2.2. validate

$ validate -w --charset=utf-8 example.html

3. Check HTML file for broken links

3.1. linkchecker

$ linkchecker -r 1 example.html
