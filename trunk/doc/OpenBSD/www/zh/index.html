十几年以来，缺省安装的系统只发现两个远程漏洞!

OpenBSD 致力于开发一个自由，基于 4.4BSD 的多平台，与 UNIX 兼容的操作系统。我们重点致力于可移植性，标准化，正确性，前瞻的安全性和完整的加密技术。OpenBSD 支持 SVR4(Solaris)，FreeBSD，Linux，BSD/OS，SunOS 和 HP-UX 下大多数程序的二进制仿真运行。

OpenBSD 可以从我们的 FTP 站点自由下载，也可以通过购买便宜的三张 CD 来获得。

当前的发行版本是 OpenBSD 4.2，发布于2007年11月1日。

OpenBSD 由志愿者开发。项目开发和发布的资金来源于贩卖 CD 和T恤，以及组织或个人的捐助。这些资金确保 OpenBSD 继续存在，并且一如既往的供所有人继续自由使用。
